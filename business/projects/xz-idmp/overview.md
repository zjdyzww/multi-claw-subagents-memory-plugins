## 项目背景

XZ-IDMP 是一个工业物联网平台，旨在为企业提供设备接入、数据采集、实时监控、智能分析和可视化展示等一体化解决方案。

## 核心功能模块

### 1. 设备管理
- 设备注册和认证
- 设备分组管理
- 设备状态监控
- 设备配置管理

### 2. 数据采集
- 支持多种协议（MQTT、HTTP、CoAP）
- 实时数据采集
- 数据清洗和转换
- 断点续传

### 3. 时序数据存储
- 基于 TDengine 的时序数据存储
- 高效的数据压缩
- 历史数据查询
- 数据聚合分析

### 4. AI 智能分析
- LLM 集成
- RAG 引擎
- 向量检索
- 智能告警

### 5. 可视化展示
- 实时数据大屏
- 历史趋势分析
- 报表导出
- 仪表盘定制

## 技术架构

```
┌─────────────────────────────────────────────────────────┐
│                    Presentation Layer                   │
│                   (Vue3 + Element UI)                   │
├─────────────────────────────────────────────────────────┤
│                      API Gateway                        │
│                    (Spring Boot)                        │
├──────────────┬──────────────┬──────────────┬───────────┤
│   Device     │    Data      │    AI        │   System  │
│   Service    │    Service   │    Service   │   Service │
├──────────────┴──────────────┴──────────────┴───────────┤
│                    Data Access Layer                    │
│              (MySQL + TDengine + Redis)                 │
├─────────────────────────────────────────────────────────┤
│                    Infrastructure                      │
│         (EMQX + Milvus + Docker Compose)               │
└─────────────────────────────────────────────────────────┘
```

## 技术决策

| 决策 | 选择 | 理由 |
|------|------|------|
| 后端框架 | Spring Boot | 生态成熟，稳定可靠 |
| 前端框架 | Vue3 | 渐进式框架，学习成本低 |
| 时序数据库 | TDengine | 高性能时序存储 |
| 消息队列 | EMQX | MQTT 协议支持优秀 |
| 向量数据库 | Milvus | AI 向量检索 |
